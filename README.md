## Requirements

* PHP 8.0 or latest
* Node 14 or latest

## How to start 

```bash
$ cp .env.example .env # setup your database
$ composer install
$ php artisan key:gen
$ npm install
$ npm run dev # optional only if any changes you make in resource/js
$ php artisan migrate --seed
$ php artisan serve
```

## Compile Assets

```bash
$ npm run dev
```

or

```bash
$ npm run watch
```
